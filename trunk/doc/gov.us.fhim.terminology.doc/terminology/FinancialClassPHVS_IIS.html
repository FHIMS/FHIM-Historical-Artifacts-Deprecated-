Definition User-defined Table 0064 - Financial class [NIP suggested values] (use in PV1-20) Financial class references a client's eligibility status at a point in time. The values in this table relate to eligibility for the Vaccine for Children (VFC) program. Local implementations may define and document local codes.
Description Note that funding source for a specific immunization is different from Financial Class. These are documented in CDC local codes table, CDCPHINVS (Value set OID 2.16.840.1.114222.4.11.3287).
