Act Consent Type

Value Set ActConsentType - 2.16.840.1.113883.1.11.19897
Code System ActCode - 2.16.840.1.113883.5.4
Source HL7
Source URL http://www.hl7.org/v3ballotarchive/v3ballot/html/infrastructure/vocabulary/vocabulary.html
Definition This value setsystem is used to specify the action referred by a consent directive. the type of consent directive, e.g., to consent or dissent to collect, access, or use in specific ways within an EHRS or for health information exchange; or to disclose health information for purposes such as research.
Description This value set is used to specify the disclosure or use action associated with a privacy consent directive. The readers are encouraged to refer to the source value set definitions for a comprehensive description and explanation of codes.
