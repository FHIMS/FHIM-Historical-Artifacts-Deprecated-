Medication Clinical Drug Name Value Set

Value Set Medication Clinical Drug Name Value Set - 2.16.840.1.113883.3.88.12.80.17
Code System RxNorm - 2.16.840.1.113883.6.88
Version 2
Definition Shall contain RxNorm normal forms for concepts type of Ingredient Name or Generic Packs. The ingredient name concepts can be found in the RxNORM file RXCONSO.RRF selecting all terms where SAB=RXNORM (selecting the normal forms), and TTY=SCD (selecting the ingredient names) or TTY=GPCK (selecting the generic packs)
