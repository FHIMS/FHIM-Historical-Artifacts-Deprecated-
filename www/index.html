• Welcome to the Federal Health Information Model (FHIM) portal. This portal is hosted by Open Health Tools (OHT) in conjunction with the Federal Health Architecture (FHA).

If you are a FHIM project member, some of the things you can do are:

  • Participate in a discussion
  • Browse and contribute source code

If you are a FHIM project owner, you can do the following:

  • Add or invite new project members
  • Create discussions
  • Edit project configurations and settings
  • Create announcements for all project members

If you are not a project member yet, request a role to get started.
